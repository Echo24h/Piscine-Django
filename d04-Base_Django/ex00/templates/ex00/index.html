Ex00 : Markdown Cheatsheet

Headers

# H1

## H2

### H3

#### H4

##### H5

###### H6

Emphasis

*italic* or _italic_

**bold** or __bold__

~~strikethrough~~

Lists

1. First ordered list item

2. Another item

- Unordered sub-list.

3. Actual numbers don't matter, just that it's a number

- Another unordered sub-list.

- You can use asterisks

- Or minuses

- Or pluses

Links

[link text](https://example.com)

Images

![alt text](image.jpg)

Blockquotes

This is a blockquote

Code

Inline `code` has `back-ticks around` it.


    // Code blocks are fenced by triple backticks
    ```javascript
    var s = "JavaScript syntax highlighting";
    alert(s);
    ```
    ```python
    s = "Python syntax highlighting"
    print(s)
    ```

Tables

Header 1 Header 2
Row 1, Col 1 Row 1, Col 2
Row 2, Col 1 Row 2, Col 2
